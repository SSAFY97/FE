# 네비바
- 최상단에 네비바가 위치해 있으며 구성 요소는 좌측부터 서비스명(LocalHub), 관광정보, 커뮤니티, chat아이콘 순서야.
- 네비바는 서비스명 / 나머지 3요소가 각각 좌측 끝, 우측 끝에서부터 나오도록 되어있어
- 네비바는 기본적으로 bg-white/50 bg-backdrop-sm 으로 해줘

# 메인 화면
- 가장 먼저 hero.webp 이미지가 최상단 배경에 그려져 있어. (배경이기 때문에 다른 레이아웃에 문제되지 않아)
- 중앙에는 관광정보 미리보기가 나와.
- 관광정보 미리보기는 탭 형식으로 되어있고 해당 탭은 (관광지, 레포츠, 문화시설, 쇼핑, 숙박, 여행코스, 축제공연행사) 로 이루어져있어.
- 각 탭을 클릭시 해당하는 내용의 이미지 그리드가 나와.
- 또한 각 탭은 5초 간격으로 다음 탭으로 자동으로 넘어가.
- 관광정보 미리보기 아래에는 🔥HOT🔥 이라는 제목으로 좋아요가 많은 상위 3개의 게시글의 제목, 작성자, 조회수, 좋아요가 선형 카드 형식으로 노출돼

# 관광 정보 화면
- 상단에 필터링과 검색을 위한 툴바가 존재해(고정 아니고 스크롤에 영향 받음)
- 툴바는 검색창, 검색 버튼, 카테고리 설정 드롭다운(전체와 데이터 7가지), 정렬 순서 드롭다운(최신순, 가나다순, 거리순) 으로 이루어져있어
- 검색 결과는 그리드(4열) 형식으로 아이템 카드가 배치돼
- 각 카드는 이미지가 상단 60%를 채우고 있고 하단에는 title과 address가 나와

# 커뮤니티 화면
- 관광정보 화면과 똑같은 툴바를 사용하고 정렬 순서 드롭다운 내용만 바꿔(최신순, 조회수 순, 좋아요 순)
- 검색 결과는 리스트 형식으로 카드가 배치돼.
- 각 카드는 좌측에 제목[댓글수]이 나오고 하단에 작게 작성자와 날짜가 있어. 우측에 좋아요, 조회수가 표시돼
