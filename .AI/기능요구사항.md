# 기술 제약 사항
- vue.js를 사용한 SPA 구조를 사용해야해
- 스타일은 tailwindCSS를 사용해줘.
- atomic design pattern을 사용해서 컴포넌트 단위로 개발해줘. (atoms, molecules, organisms, pages)
- 공통 컴포넌트에 대해 엄격히 컴포넌트화 해줘.

# 데이터
- 서울시 여행 정보 서비스를 만들거야. 데이터 형식은 첨부한 서울_관광지.json을 참고해줘
- 데이터는 서울_관광지, 서울_레포츠, 서울_문화시설, 서울_쇼핑, 서울_숙박, 서울_여행코스, 서울_축제공연행사 의 총 7가지야
- 7가지 데이터 모두 같은 json 형식을 지니고 있어. 스키마 구조는 SCHEMA.MD를 참고해줘.

# 기능
- 주어진 데이터 기반으로 
- 게시글 작성, 수정, 삭제, 게시글 목록, 게시글 상세 기능이 필요해
- 챗봇 및 챗봇과 채팅 기능이 필요해(api/chat) 
- 게시판에서 게시글 검색, 필터링 기능, 게시글 조회수 표시 기능, 게시글 좋아요 기능이 필요해
